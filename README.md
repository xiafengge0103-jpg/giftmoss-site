# GiftMoss Static Website Package

This is a GitHub Pages-ready static website package for a handmade gifts / crafts independent site.

## Included pages

- `index.html` — Homepage
- `products.html` — Product catalog with front-end category filter
- `wholesale.html` — Wholesale inquiry page
- `about.html` — About page
- `contact.html` — Contact page
- `assets/css/style.css` — Main stylesheet
- `assets/js/main.js` — Mobile menu, product filter and demo form success message
- `assets/img/` — Original SVG demo product images and logo

## How to deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to repository `Settings` → `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save. GitHub will provide a public website link.

## What to edit before publishing

- Replace `GiftMoss` with your own brand name.
- Replace demo email and WhatsApp in all pages.
- Replace SVG demo images with your real product photos if needed.
- Connect forms to a real service such as Formspree, Netlify Forms, Shopify app, WordPress plugin, or your own backend.
- Update product descriptions, MOQ, material, packaging and certifications according to your real supply chain.

## Notes

This package uses pure HTML, CSS and JavaScript. It does not require Node.js, npm, database or server-side code.
